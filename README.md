# ki

this is tree like application written in Go

# Install

```bash
$ go get -u github.com/pankona/ki
```

# Usage

```bash
# Specify directory (s) to show directory tree
$ ki ./hoge
./
|-- fuga/
|-- piyo.txt
`-- zzz/
    |-- piyo.txt
    `-- qqq.txt
```
